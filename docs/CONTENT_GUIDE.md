# Content Guide

## Brand Voice

All website content should feel:

- calm
- precise
- technical
- confident
- premium

The tone should resemble:

- Apple product communication
- Linear technical clarity
- Vercel engineering confidence

Avoid aggressive marketing language.

---

# Positioning

The brand:

</D> Professor Danya

Position:

AI Engineer building automation systems, intelligent tools and digital products.

Core principle:

"Build less. Build better."

---

# Writing Principles

Every piece of content should answer:

1. What was built?
2. Why was it built?
3. How does it work?
4. What value does it create?

Prefer clarity over exaggeration.

---

# Avoid

Do not use:

- revolutionary
- game-changing
- next generation
- cutting-edge AI solutions
- world-class
- guaranteed results

Avoid empty marketing phrases.

---

# Prefer

Instead of:

"I create amazing websites and powerful AI solutions."

Use:

"I build AI-powered systems, automation workflows and digital products."

---

Instead of:

"We help businesses grow with innovative technology."

Use:

"We design and build systems that reduce repetitive work and connect business processes."

---

# Project Content Rules

Each project should include:

## Context

What problem existed?

## Solution

What system was created?

## Engineering

What technologies and architecture were used?

## Role

What was personally designed and developed?

## Status

Use:

- Prototype
- Production
- Experiment

---

# Metrics Policy

Never invent:

- users
- revenue
- clients
- performance numbers
- business results

Only use verified information.

If no metrics exist:

Explain the system and engineering challenge instead.

---

# Technology Presentation

Technology should support the story.

Do not make projects look like:

React + Next.js + Python lists.

Show:

- systems
- workflows
- integrations
- architecture
- decisions

---

# Calls To Action

Preferred:

- View project
- Explore system
- Contact
- Discuss a project

Avoid:

- Buy now
- Get started today
- Limited offer
