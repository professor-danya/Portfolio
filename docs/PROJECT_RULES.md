PROJECT RULES

Project Goal

Build an ultra-premium personal portfolio inspired by Apple, Linear, Vercel, Stripe and Porsche.

The final result must feel expensive, minimalistic, futuristic and extremely polished.

Never generate template-looking layouts.

⸻

Stack

* Next.js App Router
* React
* TypeScript
* Tailwind CSS
* Framer Motion

⸻

Design Language

Style:

Apple × Linear × Vercel × Porsche

Dark graphite theme.

No colorful gradients everywhere.

One premium accent color only.

Large whitespace.

Perfect spacing.

Minimal UI.

Expensive feeling.

⸻

Components

Always reuse components.

Never duplicate code.

Every UI element should be reusable.

⸻

Hero

The hero must be cinematic.

Large typography.

Strong hierarchy.

Custom background.

Animated lighting.

Premium buttons.

⸻

Projects

Projects must never be simple text cards.

Each project should look like a real product showcase.

Large previews.

Beautiful hover animations.

Depth.

⸻

Animations

Use Framer Motion.

Animations must be subtle.

No exaggerated movement.

Respect prefers-reduced-motion.

⸻

Performance

Performance is critical.

Avoid unnecessary re-renders.

Avoid heavy mouse tracking.

Optimize images.

Lazy load when possible.

⸻

Code Style

Write production-ready code.

Strict TypeScript.

Clean architecture.

Readable naming.

Small reusable components.

⸻

Never

Never create generic portfolio templates.

Never use random gradients.

Never overuse glassmorphism.

Never reduce readability.

Never sacrifice performance for animations.

⸻

Goal

Create a portfolio that looks like it belongs to a senior AI engineer and premium digital product studio.

## Development Rules

Before creating new components:

- Analyze existing architecture
- Prefer reusable components
- Avoid unnecessary dependencies
- Keep TypeScript strict
- Do not modify unrelated sections

---

## Product Philosophy

The portfolio should demonstrate:

- real engineering ability
- AI systems thinking
- automation expertise
- product design understanding

Not:

- generic frontend templates
- fake SaaS claims
- marketing hype
